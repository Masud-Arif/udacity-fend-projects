Who Am I?

My name is Matthew. I strive to build elegant, modern, highly functional websites. Let me know how I'm doing if you have time.

I began as a web developer in mid-2015 by joining a relatively new program called FreeCodeCamp. After a few months I felt that I'd reached the end of FreeCodeCamp's thoroughly developed content, and transitioned to a nanodegree program offered by Udacity. So far I've found Udacity's program to offer a greater depth of content, as well as much needed access to professional reviews of my work.

I have experience programming in various languages including Ruby, Python, C#, and JavaScript. I've worked with jQuery, AngularJS, Polymer, Node.js, Gulp, Grunt, Bower, Vagrant, and other web development technologies. I plan to bring my front-end development skills up to par with what's expected of industry professionals by taking full advantage of Udacity's nanodegree program.

I have a bachelor's degree in Security Management and am currently employed full-time in my field. I've grown fundamentally dissatisfied with my career choice and am in the process of transforming my programming hobby into a programming career. If you'd like to help, have a look at my work and give me some feedback. I'm always looking to improve. Thanks for reading!

Featured Work

Escape By Water!

January 2016

A clone of the popular frogger game built using a barebones game engine provided by Udacity. I worked heavily with the HTML5 canvas element in this project, and used JavaScript inheritance, the browser's local storage, and the Web Audio API while implementing various features. The game features a persistent scoreboard, menus, canvas-drawn buttons, and mobile support.

Pomodoro Clock

September 2015

Based on the Pomodoro Technique, this site provides a time management clock with customizable work and break cycles. I used the canvas element to programmatically draw the clock itself, and used a browser notification combined with an audible alarm to alert the user of the end of a clock cycle.

Local Weather

August 2015

Uses your browser's location services to display local weather information gathered from Yahoo's weather API. If you happen to be in the USA, it also collects information from epa.gov's API to display the UV index high for the day. It supports manual location searching and both imperial and metric units of measure. Also included is a dynamic background based on time and weather conditions.

Quote Machine

July 2015

Displays a random quote from a collection. The challenge was in providing a tweet button that updates with the currently displayed quote instead of constantly using the first displayed quote, then handling cases where the quote exceeds Twitter's character limit.
