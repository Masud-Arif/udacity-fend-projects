Who Am I?

My name is Matthew. I strive to build elegant, modern, highly functional websites. Let me know how I'm doing if you have time.

I began as a web developer in mid-2015 by joining a relatively new program called FreeCodeCamp. After a few months I felt that I'd reached the end of FreeCodeCamp's thoroughly developed content and transitioned to a nanodegree program offered by Udacity. At the time of my transition I found Udacity's program to offer a greater depth of content, as well as much needed access to professional reviews of my work.

I have experience programming in various languages. I've worked with a number of web development technologies, and am always eager to learn about new things. I plan to bring my front-end development skills up to par with what's expected of industry professionals by taking full advantage of Udacity's nanodegree program.

I have a bachelor's degree in Security Management and am currently employed full-time in my field. I've grown fundamentally dissatisfied with my career choice and am in the process of transforming my programming hobby into a programming career. If you'd like to help, have a look at my work and give me some feedback. I'm always looking to improve. Thanks for reading!

Featured Work

Escape By Water!

January 2016

A clone of the popular frogger game built using a barebones game engine provided by Udacity. I worked heavily with the HTML5 canvas element in this project, and used JavaScript inheritance, the browser's local storage, and the Web Audio API while implementing various features. The game features a persistent scoreboard, menus, canvas-drawn buttons, and mobile support.

Resume

December 2015

My online resume for web development. It dynamically creates page content with data retrieved from a JSON object and utilizes CSS transitions, a Polymer element, and Google Maps' API to provide additional functionality. I also expanded my workflow options with this project by learning to use Bower and by refining my usage of Gulp.

Pomodoro Clock

September 2015

Based on the Pomodoro Technique, this site provides a time management clock with customizable work and break cycles. I used the canvas element to programmatically draw the clock itself, and used a browser notification combined with an audible alarm to alert the user of the end of a clock cycle.

July 2015

My first project. It interfaces with twitch.tv's public API to retrieve and display information on a list of streamers. Provides options for sorting and filtering. Also included in the Github repository is an incomplete but functional AngularJS refactor, in which I learned AngularJS best practices by following John Papa's Angular Style Guide.
